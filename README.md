# GenAI-Code-Reviewer-Python

## 💻 Installation & Usage

1.  Clone the Repository
   
    **git clone " https://github.com/SURESHKUMAR-007/GenAI-Code-Reviewer-Python.git "
    cd GenAI-Code-Reviewer-Python**
    
2.  Install Dependencies
   
    **pip install streamlit**
    
3.  Run the Application
   
    **streamlit run genai_code_reviewer.py**
    
    *(If you get a path error, try: **python -m streamlit run genai_code_reviewer.py**)
